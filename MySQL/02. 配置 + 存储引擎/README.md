### InnoDB和MyISAM对比

| 对比项 | MyISAM                                                 | InnoDB                                     |
| ------ | ------------------------------------------------------ | ------------------------------------------ |
| 主外键 | 不支持                                                 | 支持                                       |
| 事务   | 不支持                                                 | 支持                                       |
| 行表锁 | 表锁；即使操作一条记录也会锁住整张表，不适合高并发操作 | 行锁；操作时只会锁住某一行，适合高并发操作 |
| 缓存   | 只缓存索引，不缓存真实数据，内存要求小                 | 索引和真实数据都可以缓存，内存要求高       |
| 表空间 | 小                                                     | 大                                         |
| 关注点 | 性能                                                   | 事务                                       |
